Faq

General

Register User?
Call API - http://bccwallet.io/bcc/wallet/apiindex.php?method=userRegistration

INPUT PARAMETERS- {
"email_id":"tempclient@temop.com"
"user_password" : "12345678",
"confirm_password" : "12345678",
"spending_password" : "12345678",
"confirm_spending_password":"12345678"
}
Login User?
Call API - http://bccwallet.io/bcc/wallet/apiindex.php?method=Login

INPUT PARAMETERS- {
"email_id" : "tempclient@temop.com",
"password" : "1234567"
}
Update Spending Password?
Call API- http://bccwallet.io/bcc/wallet/apiindex.php?method=updateSpendingPassword

INPUT PARAMETERS- {
"email_id" : "tempclient@temop.com",
"spending_password" : "1234567",
"confirm_spending_password":"1234567"
}
Generate new address?
Call API - http://bccwallet.io/bcc/wallet/apiindex.php?method=getnewaddress

INPUT PARAMETERS- {
"email_id" : "tempclient@temop.com"
}
Send BCC?
Call API - http://bccwallet.io/bcc/wallet/apiindex.php?method=sendCoin

INPUT PARAMETERS- {
"email_id" : "tempclient@temop.com",
"wallet_address" : "TDo6iTruH8pmuePFJJr1fWAfGkAN1ebbpH",
"trans_amount" : "5",
"spending_password":"1234567"
}
Track Transactions?

INPUT PARAMETERS- {

}
